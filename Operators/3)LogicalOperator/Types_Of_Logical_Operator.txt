/*LOGICAL OPERATORS 
->Logical operators are used to perform logical operations on boolean values.
->They are commonly used in conditional statements and loops to combine multiple conditions and make decisions based on 
  the truthiness of those conditions.
->used to perform logical operations on boolean values
->used to combine multiple conditions


->TYPES OF LOGICAL OPERATORS:

A)LOGICAL AND (&&):
->The logical AND operator (&&) returns true if both operands are true; otherwise, it returns false.    
for example: 
result|statemnt1|statement2|
TRUE  |  TRUE   |  TRUE    | 
FALSE |  FALSE  |  TRUE    | 
FALSE |  TRUE   |  FALSE   |
FALSE |  FALSE  |  FALSE   |

B)LOGICAL OR (||):
->The logical OR operator (||) returns true if at least one of the operands is true; otherwise, it returns false.
for example:
result|statemnt1|statement2|
TRUE  |  TRUE   |  TRUE    | 
TRUE  |  FALSE  |  TRUE    | 
TRUE  |  TRUE   |  FALSE   |
FALSE |  FALSE  |  FALSE   |

C)LOGICAL NOT (!):
->The logical NOT operator (!) is a unary operator that negates the value of a boolean expression.
for example:
result|statement|
TRUE  |  FALSE  |  
FALSE |  TRUE   |

*/
